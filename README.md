**Social Signal Prototype Overview**
Social Signal Prototype is a cybersecurity-focused tool designed to detect potentially harmful or coordinated anti-India messaging on social media platforms.

The system analyzes social media text and flags posts containing suspicious keywords, phrases, or obfuscated text patterns commonly used in propaganda or influence campaigns.

This project demonstrates a simple pipeline for:

Social media data collection
Text normalization
Keyword and pattern detection
Basic analytics and visualization
